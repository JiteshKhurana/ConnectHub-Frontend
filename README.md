# Thapar Events

## TechStack

- TypeScript
- React.js
- Redux - State Management
- Tailwind CSS - UI
- Shadcn - UI
- React Hook Form - Forms
- Zod - Form Validation
- React Router DOM - Routing
- Vite - Bundler
- Axios
- React Big Calender
